People and Projects

An incomplete list of nicknames on #proglangdesign and their labours of love. Some are definitely missing - just make a commit to the Github repository with the changes you want to see.
Athas
Futhark, a high-performance parallel functional array language targeting GPUs.
Boney
Plasma, which aims to strike a balance between functional and imperative programming. Previously worked on Mercury a logic language.
Cheery
Lever, a dynamically typed language built to absorb features from other languages.
cjh`
Icarus is a minimum-viable type-safe imperative language designed to serve as a platform for exploring mutation and general side-effect control.
nee
Fox is an ever-evolving experiment in declarativeness that strives to one day settle on a consistent syntax.
yumaikas
PISC attemps to make concatenative programming less mind-bendy via pervasive use of locals, closures, the smallest set of syntax hacks to make code look nice.

How do I join the IRC channel?

  • Install one of the (many!) available IRC clients
  • Use riot.im (recommended by HackerFoo, join #freenode_#proglangdesign:matrix.org)
  • Use Freenode's web-based client. Join the #proglangdesign channel.
